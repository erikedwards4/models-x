# models-x
Repo for external models in JAX.  

See repo model-ext for external models in Pytorch.  
See repo models-int for internally-developed models.  

This is a library of classic, well-known and interesting models available  
externally at model hubs such as HuggingFace, but herein implemented from  
scratch following best-practices and giving low-level control.  

This also includes some paper-to-code models specified in a publication  
and perhaps given in an open-source repo.  

See the src/models_x directory! Lots of models with system diagrams  
made using draw.io (see the png folder for the pictures). Each model  
has a README with some points, citations, and external links.
